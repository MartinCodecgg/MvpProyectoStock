# MVP Web - Sistema de Gestión de Stock

Sistema de gestión de stock para ferreterías con funcionalidad de punto de venta y actualización de inventario mediante fotografías.

## Requisitos

- Node.js 18 o superior
- NPM o equivalente

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:5173`

## Build

```bash
npm run build
```

## Tecnologías

- **Frontend**: Svelte 5, TypeScript
- **Estilos**: Tailwind CSS
- **Build**: Vite
- **Modo**: Solo Frontend (datos simulados, sin backend)

## Características

- ✅ Punto de venta (Caja)
- ✅ Gestión de stock
- ✅ Actualización de inventario por foto (simulado)
- ✅ Temas claro y oscuro
- ✅ Diseño optimizado para desktop
